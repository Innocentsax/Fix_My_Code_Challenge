# README

Blog application:

## How to install it:

- `$ gem install bundler`
- `$ bundle install`
- `$ rails db:migrate RAILS_ENV=development`

## How to run the server

`$ rails s -b 0.0.0.0 -p 5000`

## How to start the rails console

`$ rails c`
